# E-report_v0.0.1

Sistem Manajemen Konsultasi (E-Konsul) — CRM untuk mengelola prospek klien, sesi konsultasi, dan analitik bisnis.

## Tech Stack
- **Backend**: Laravel 11 / PHP 8.2
- **Frontend**: Blade + Tailwind CSS + Alpine.js + DaisyUI
- **Build Tool**: Vite
- **Database**: MySQL

## Fitur Utama
- Manajemen Konsultasi & Lead
- Multi-Akun & Role-Based Access (Super Admin / Admin)
- Notes & Reminders per Konsultasi
- Dashboard Analitik & Laporan
- Sistem Notifikasi (Polling)
- Audit Trail Log
- Master Data Management
- Import/Export CSV
- Report Attendance
